Methodology

Our approach combines active exploration with unsupervised skill discovery to create a comprehensive framework for robotic manipulation learning. The methodology consists of several key components that work together to maximize learning efficiency.

Active Exploration Framework

The active exploration component leverages Fisher Information [14] to guide the robot toward maximally informative trajectories. This is achieved through the optimization of the following objective:

I(θ) = E[∇_θ log p(τ|θ) ∇_θ log p(τ|θ)^T]

Where θ represents the system parameters and τ denotes the trajectory. This formulation ensures that the robot explores regions of the parameter space that provide the most information about the underlying dynamics [15].

Key Features:

  • Adaptive parameter selection based on information gain
  • Real-time trajectory optimization
  • Integration with existing sim-to-real transfer methods
  • Scalable to high-dimensional parameter spaces

Implementation Steps:

  1. Initialize parameter distribution P(θ)
  2. Generate candidate trajectories using current policy
  3. Evaluate Fisher Information for each trajectory
  4. Select trajectory with maximum information gain
  5. Execute trajectory and update parameter estimates
  6. Repeat until convergence or maximum iterations

Unsupervised Skill Discovery

Building upon the active exploration framework, we extend the approach to discover diverse manipulation skills without task-specific supervision. The skill discovery process operates on the principle of maximizing mutual information between skills and states.

The skill discovery algorithm learns a diverse set of behaviors by encouraging the agent to visit different regions of the state space. This is achieved through a reward function that balances exploration and exploitation, ensuring that the learned skills are both diverse and useful for downstream tasks.

Each discovered skill corresponds to a specific manipulation primitive that can be combined with other skills to perform complex tasks. The diversity of skills is measured using a distance metric in the state space, ensuring that each skill explores a unique region of the manipulation workspace.

Introduction

Robotic learning in simulation has become an indispensable tool for accelerating research and reducing costs before deployment in the real world [1]. Training in simulation provides safety, scalability, and access to large amounts of interaction data. However, the policies that perform well in simulation often degrade significantly once transferred to real robots - a challenge commonly referred to as the reality gap [2]. This gap arises from mismatches between the simplified models used in simulation and the complex, noisy dynamics of physical systems.

A major line of research in sim-to-real transfer focuses on mitigating this gap. Two prominent strategies are Domain Randomization (DR) [3] and System Identification (SI) [4]. DR aims to expose policies to a wide variety of randomized environments during training, encouraging robustness to unseen real-world conditions. SI, on the other hand, attempts to explicitly identify the dynamics of the target system, either before or during deployment, and adjust the simulation parameters accordingly. While DR offers generalization at the expense of sample efficiency, SI provides more targeted adaptation but often suffers from insufficient or uninformative data for accurate identification.

The ASID (Active exploration for System Identification in robotic manipulation) framework [5] addresses this limitation by proposing methods for actively selecting exploratory behaviors that maximize information about unknown system parameters. Rather than passively collecting data or relying on heuristic exploration, ASID leverages an optimization criterion based on Fisher Information to generate trajectories that are maximally informative for system identification. This approach aims to make SI more efficient and reliable, ultimately improving sim-to-real transfer in robotic manipulation tasks.

Beyond system identification, such objectives resonate with the broader idea of curiosity-driven exploration - encouraging robots to explore their environment without being explicitly told what to do [6]. Other approaches have pursued this direction by aiming to visit as much of the state space as possible, under the intuition that wider coverage should reveal more skills [7]. However, merely covering the state space does not guarantee that the learned skills are meaningful or useful for downstream tasks.

This brings us to the complementary problem of acquiring general-purpose behaviors that can accelerate the learning of new downstream tasks. Instead of training from scratch for every new objective, agents can benefit from a repertoire of skills - temporally extended action patterns that serve as primitives for hierarchical reinforcement learning [8]. The paper "Diversity is All You Need" (DIAYN) [9] introduced the idea of unsupervised skill discovery, where an agent learns a diverse set of behaviors without requiring an external reward function. These unsupervised skills not only provide a foundation for faster task learning but also enhance exploration in environments with sparse or delayed rewards, and can support applications such as imitation learning and transfer across related tasks [10].

However, while DIAYN and its successors demonstrated compelling results in locomotion domains such as cheetah or humanoid agents [11,12], robotic manipulation remains a largely unsolved challenge [13]. Current methods struggle to produce diverse and reusable manipulation skills without task-specific supervision, leaving open the question of how to adapt unsupervised exploration principles to this more complex setting.

Our project builds on this motivation by seeking to extend the ASID framework beyond system identification. ASID showed how the Fisher Information can guide robots toward collecting trajectories that are informative for parameter inference. We propose to adapt this principle for unsupervised skill discovery: instead of treating the objective as purely about maximizing entropy or covering all parameters equally, we reinterpret simulated parameters as a curiosity signal. By selectively focusing on information along certain parameter axes while ignoring others, the agent can discover a richer and more diverse set of skills. This perspective opens the door to curiosity-guided skill discovery for robotic manipulation, bridging the gap between system identification for sim-to-real transfer and unsupervised exploration for skill learning.

Key Contributions

  • Extension of ASID framework for unsupervised skill discovery in robotic manipulation
  • Novel information-theoretic approach to curiosity-driven exploration
  • Comprehensive evaluation across multiple manipulation tasks and environments
  • Demonstration of improved sim-to-real transfer performance compared to baseline methods

Experimental Validation

  1. Simulation experiments with varying physical parameters and task complexities
  2. Real-world validation on physical robotic manipulation platforms
  3. Comparison with state-of-the-art domain randomization and system identification methods
  4. Analysis of skill diversity and transfer performance metrics
  5. Ablation studies to understand the contribution of each component
